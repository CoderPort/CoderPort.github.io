DevTick

DevTick is a productivity web app for developers. It includes task management, project categorization, a developer-specific contact book, API key manager, project idea vault, and built-in AI-powered chat for assistance. DevTick helps you stay organized while working on what matters.

View on GitHub

Flask Development Template

The Flask Development Template is a foundational template designed to help developers kickstart their Flask web applications. It includes structured project layouts, essential configurations, SQLite integration, authentication, and more to streamline the development process.

CodeSpark Initiative

CodeSpark is our community outreach program where we visit middle schools and educational events to ignite passion for computer science. Through interactive workshops, hackathon mentoring, and career talks, we introduce students to the creative possibilities of coding and help bridge the gap between education and real-world tech innovation.
